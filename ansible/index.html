Image server for Incus and LXC

This domain hosts a public image server for use by Incus and LXC.

All images available on this server are generated using distrobuilder along with our YAML definitions files.
The build themselves can be seen on our Jenkins server.

Images are published after basic automated functionality testing.
Should an image regress in some way, images of the past 3 days are kept available.

In LXC, this image server can be used by selecting the "lxc-download" template.
In Incus, this image server is reachable through the "images:" default remote
